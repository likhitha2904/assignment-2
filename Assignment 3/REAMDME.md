# AI API Integration Project

## Project Description

This project demonstrates how to integrate multiple Generative AI APIs using Python. The objective is to build individual programs that query different AI providers and a unified application that allows users to interact with multiple AI systems through a single interface.

The project integrates the following AI platforms:

* OpenAI
* Groq
* Ollama (local AI model)
* Hugging Face
* Google Gemini
* Cohere

Each API is accessed through a dedicated Python script. Additionally, a multi-API interface allows users to select a provider, submit prompts, and receive AI-generated responses.

To enhance functionality and usability, the project also includes:

* Multi-API query interface
* Streamlit graphical user interface
* Persistent conversation history with timestamps
* Response comparison between different AI providers
* Streaming responses
* Error handling for API failures

This project demonstrates practical experience working with modern AI services and building applications that combine multiple AI providers in a single system.

# Description of Each File

## openai_example.py

This script demonstrates how to interact with OpenAI models using the OpenAI API.
It takes a user prompt, sends it to an OpenAI model (gpt-4o-mini), and displays the generated response.
It also includes error handling to manage API quota or request failures.

---

## groq_example.py

This program connects to the Groq API and queries a LLaMA-based model (llama-3.1-8b-instant).
Groq provides very fast inference for open-source models.
The script sends a prompt to the Groq model and prints the generated response.

---

## ollama_example.py

This script demonstrates how to run AI models locally using Ollama.
Instead of using a cloud API, the model(llama3) runs directly on the user's computer.
The program sends prompts to the locally running model through the Ollama API.

---

## huggingface_example.py

This program shows how to interact with models hosted on Hugging Face using an API token.
The script sends a prompt to a Hugging Face model(meta-llama/Llama-3.1-8B-Instruct) and returns the generated output.

---

## gemini_example.py

This script demonstrates how to use Google's Gemini API.
The program sends a prompt to a Gemini model(gemini-2.0-flash) and prints the AI-generated response.

---

## cohere_example.py

This program interacts with Cohere's language models using their Chat API.
The script sends a prompt and retrieves a response generated by the Cohere model(command-a-03-2025).

---

## multi_api_query.py

This program provides a unified interface to interact with multiple AI providers.
Users can select the desired AI provider and enter a prompt.
The program then calls the appropriate API function and displays the response.

---

## app.py

This file implements a Streamlit-based graphical user interface for the project. (enhanced version of multi_api_query.py)
The interface allows users to:

* Select an AI provider
* Enter prompts
* View responses
* Compare outputs from multiple APIs (compare speed, word length, character length and if the output is structured or not.)
* Store persistent conversation history along with time stamp.
* Display streaming responses (only for groq.)

# Environment Variables

Create a `.env` file in the project directory:

```env
OPENAI_API_KEY=your_openai_key
GROQ_API_KEY=your_groq_key
HUGGINGFACE_API_KEY=your_hf_key
GOOGLE_API_KEY=your_gemini_key
COHERE_API_KEY=your_cohere_key
```

Important:
The `.env` file should **not be committed to GitHub**.

# How to Obtain API Keys

## OpenAI

1. Visit https://platform.openai.com
2. Sign up or log in
3. Go to **API Keys**
4. Create a new API key

---

## Groq

1. Visit https://console.groq.com
2. Create an account
3. Navigate to **API Keys**
4. Generate a new key

---

## Hugging Face

1. Visit https://huggingface.co
2. Sign up
3. Go to **Settings → Access Tokens**
4. Create a **Read Token**

---

## Google Gemini

1. Visit https://aistudio.google.com/app/apikey
2. Create a new API key
3. Copy the key into the `.env` file

---

## Cohere

1. Visit https://dashboard.cohere.com
2. Create an account
3. Generate an API key from the dashboard

---

## Ollama (Local Model)

1. Download Ollama from:
   https://ollama.ai

2. Pull a model:

```bash
ollama pull llama3
```

3. Run the model:

```bash
ollama run llama3
```
